PDF文件按页分割工具使用说明：

当前目录使用命令行或批处理执行

java -cp ".;lib/*" com.snail.tools.pdf.helper.PDFSpliter 参数1 参数2 参数3

例

"E:\note\book\重构_改善既有代码的设计[高清版]" 4 +

参数1【必选】：pdf文件完整路径，不能包含后缀(.pdf)，如果路径有空格等特殊符号需要用双引号引用。
参数2【必选】：要按页分割的数量。
参数3【可选】：“+”表示如果分割数量不能被总页数整除时，剩余的页会单独生成一个尾文件(文件名以_tail结尾)；否则都追加到最后一个分割文件中。
